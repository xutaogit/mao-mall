# 后台管理系统使用说明

## 🚀 快速启动

### 1. 启动后端服务
```bash
cd /Users/sam/project/mao-mall/backend
npm run dev
```
后端服务运行在：http://localhost:3001

### 2. 启动后台管理系统
```bash
cd /Users/sam/project/mao-mall/admin
npm run dev
```
或使用启动脚本：
```bash
cd /Users/sam/project/mao-mall/admin
./start.sh
```

后台管理系统运行在：http://localhost:5174

## 📝 功能说明

### 商品管理
- **商品列表**：查看所有商品，支持搜索、上架/下架
- **添加商品**：点击"添加商品"按钮，跳转到编辑页面
- **编辑商品**：点击"编辑"按钮，跳转到编辑页面
  - 基本信息：名称、分类、编号、价格、库存
  - 商品图片：主图 + 相册（最多5张）
  - SKU 规格：支持多规格商品（颜色、尺寸等）
  - 商品详情：描述、HTML 详情

### 分类管理
- 查看分类列表
- 添加/编辑/删除分类
- 支持多级分类

### 订单管理
- 查看订单列表
- 订单详情
- 订单状态管理

### 退款管理
- 查看退款申请列表
- 审核退款（通过/拒绝）
- 查看退款详情

### 优惠券管理
- 创建优惠券（满减券/折扣券/无门槛券）
- 编辑/删除优惠券
- 查看优惠券使用情况

## 🔧 常见问题

### 1. 编辑商品时提示错误

**可能原因：**
- 后端服务未启动
- 端口配置错误
- API 响应格式不匹配

**解决方案：**
1. 确保后端服务运行在 3001 端口
2. 检查 `vite.config.js` 中的代理配置
3. 检查浏览器控制台的错误信息

### 2. 图片上传失败

**说明：**
当前版本使用本地预览（Base64），图片不会真正上传到服务器。

**后续优化：**
- 集成云存储服务（阿里云 OSS/腾讯云 COS）
- 或实现本地文件上传接口

### 3. SKU 列表不显示

**可能原因：**
- 规格名称或规格值为空
- 未点击"添加规格"按钮

**解决方案：**
1. 确保规格名称和规格值都已填写
2. 规格值支持多选，按回车添加

## 📊 数据格式

### 商品数据结构
```javascript
{
  name: '商品名称',
  categoryId: '分类ID',
  productSn: '商品编号',
  price: 99.00,
  stock: 100,
  pic: '主图URL',
  albumPics: ['图片1', '图片2'],
  specs: [
    { name: '颜色', values: ['红色', '蓝色'] },
    { name: '尺寸', values: ['S', 'M', 'L'] }
  ],
  skuList: [
    { 
      颜色: '红色', 
      尺寸: 'S', 
      price: 99, 
      stock: 10,
      skuCode: 'SKU001'
    }
  ]
}
```

### 优惠券数据结构
```javascript
{
  name: '优惠券名称',
  type: 0,  // 0-满减券 1-折扣券 2-无门槛券
  amount: 10,  // 优惠额度
  minPoint: 100,  // 使用门槛
  count: 1000,  // 发放数量
  perLimit: 1,  // 每人限领
  startTime: '2026-01-01',
  endTime: '2026-12-31'
}
```

## 🎯 开发建议

### 1. 图片上传优化
建议集成云存储服务，避免使用 Base64 存储图片。

### 2. 富文本编辑器
建议集成富文本编辑器（Quill/TinyMCE）来编辑商品详情。

### 3. 批量操作
添加批量上架/下架、批量修改价格等功能。

### 4. 数据导入导出
支持 Excel 导入导出商品数据。
